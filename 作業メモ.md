## 目的
SKILLSセクションに遊び心を加えるため、各技術のアイコン画像を挿入する。

## 配置場所
`index.html`のSKILLSセクション内、各スキル名の前に配置する。

**例:**
```html
<li><img src="images/cpp_icon.png" alt="C++ icon" class="skill-icon"> C++ <span class="stars">★★★☆☆</span></li>
```

## 画像フォルダ
プロジェクトルートに`images`フォルダを作成し、上記の画像ファイルを格納する。
（注：実際の画像ファイルは別途用意する必要があります）
